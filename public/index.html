Brief vidéo ~10 minutes

Indiquez un domaine et un angle : vous obtenez un plan JSON (chapitres, durées, points à dire, idées de B-roll). Activez le découpage en petits clips pour tourner ou exporter bloc par bloc, puis les coller sur une timeline — idéal pour fabriquer une grande vidéo sans tout enregistrer d’un seul tenant.

Formulaire

10 min

API directe : GET /api/plan (aide) ou POST /api/plan avec Content-Type: application/json.

Résultat

en attente 

{}
Déployé sur Vercel : connectez ce dépôt, laissez les réglages par défaut (pas de commande de build obligatoire). Le dossier public/ sert l’interface ; api/plan.py expose l’API Python. Pour un MP4 brouillon (diaporama) à partir du JSON : python3 -m video_agent.slideshow --json brief.json -o video.mp4 en local (FFmpeg requis).
